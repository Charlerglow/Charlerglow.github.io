第七章 卤代烃

一、卤代烃的分类和命名

1. 卤代烃的定义

烃分子中一个或多个氢原子被卤原子（F\ce{F}F、Cl\ce{Cl}Cl、Br\ce{Br}Br、I\ce{I}I）取代的化合物，官能团为卤原子，通式为 R−X\ce{R-X}R−X（或芳基卤代烃 Ar−X\ce{Ar-X}Ar−X）。氟代烃因制法、性质特殊，单独讨论。

2. 分类

（1）按烃基结构分类

类别 细分类型 定义 示例
饱和卤代烃 - 饱和烃（烷烃/环烷烃）的氢被卤原子取代 氯乙烷（CHX3CHX2Cl\ce{CH3CH2Cl}CHX3​CHX2​Cl）、溴环己烷（CX6HX11Br\ce{C6H11Br}CX6​HX11​Br）
不饱和卤代烃 乙烯型 卤原子直接连双键碳（sp2sp^2sp2 杂化） 氯乙烯（CHX2=CHCl\ce{CH2=CHCl}CHX2​=CHCl）
烯丙型 卤原子与双键隔 1 个饱和碳 烯丙基氯（CHX2=CHCHX2Cl\ce{CH2=CHCH2Cl}CHX2​=CHCHX2​Cl）
隔离型 卤原子与双键隔 2 个及以上饱和碳 4-氯丁-1-烯（CHX2=CHCHX2CHX2Cl\ce{CH2=CHCH2CH2Cl}CHX2​=CHCHX2​CHX2​Cl）
卤代芳烃 苯基型 卤原子直接连苯环碳（sp2sp^2sp2 杂化） 氯苯（CX6HX5Cl\ce{C6H5Cl}CX6​HX5​Cl）
苄基型 卤原子与苯环隔 1 个饱和碳 苄基溴（CX6HX5CHX2Br\ce{C6H5CH2Br}CX6​HX5​CHX2​Br）
隔离型 卤原子与苯环隔 2 个及以上饱和碳 3-苯基-1-氯丙烷（CX6HX5CHX2CHX2CHX2Cl\ce{C6H5CH2CH2CH2Cl}CX6​HX5​CHX2​CHX2​CHX2​Cl）

（2）按卤原子连接碳原子类型分类

  • 伯卤代烃（1°）：卤原子连伯碳（R−CHX2−X\ce{R-CH2-X}R−CHX2​−X），如 1-氯丙烷（CHX3CHX2CHX2Cl\ce{CH3CH2CH2Cl}CHX3​CHX2​CHX2​Cl）；
  • 仲卤代烃（2°）：卤原子连仲碳（R−CH(X)−RX′\ce{R-CH(X)-R'}R−CH(X)−RX′），如 2-溴丁烷（CHX3CHBrCHX2CHX3\ce{CH3CHBrCH2CH3}CHX3​CHBrCHX2​CHX3​）；
  • 叔卤代烃（3°）：卤原子连叔碳（RX3C−X\ce{R3C-X}RX3​C−X），如叔丁基氯（(CHX3)X3CCl\ce{(CH3)3CCl}(CHX3​)X3​CCl）。

3. 命名

（1）普通命名法（习惯命名法）

  • 规则：“烃基 + 卤原子”，直接命名简单卤代烃。
  • 示例：异丙基溴（(CHX3)X2CHBr\ce{(CH3)2CHBr}(CHX3​)X2​CHBr）、叔丁基氯（(CHX3)X3CCl\ce{(CH3)3CCl}(CHX3​)X3​CCl）、烯丙基溴（CHX2=CHCHX2Br\ce{CH2=CHCH2Br}CHX2​=CHCHX2​Br）。
  • 俗名：多卤代烷常用俗名，如 CHClX3\ce{CHCl3}CHClX3​（氯仿）、CHBrX3\ce{CHBr3}CHBrX3​（溴仿）、CHIX3\ce{CHI3}CHIX3​（碘仿）、CClX4\ce{CCl4}CClX4​（四氯化碳）。

（2）系统命名法

  • 规则：以烃为母体，卤原子为取代基，编号遵循“位次最小”原则。
    • 卤代烷：选含卤原子的最长碳链，如 CHX3CH(Cl)CHX2CHX3\ce{CH3CH(Cl)CH2CH3}CHX3​CH(Cl)CHX2​CHX3​（2-氯丁烷）；
    • 卤代环烷：环为母体，卤原子、烷基为取代基，如 1-甲基-4-溴环己烷（结构用图表示更合适）；
    • 卤代烯烃：选含双键和卤原子的最长碳链，双键编号优先，如 CHX3CH(Br)CH=CHX2\ce{CH3CH(Br)CH=CH2}CHX3​CH(Br)CH=CHX2​（3-溴丁-1-烯）；
    • 卤代芳烃：苯环为母体，卤原子、烷基为取代基（具体结构建议配图），如 4-溴-2-氯乙基苯（仅示规则）。

二、卤代烃的制法

1. 烃的卤化

  • 烷烃自由基卤化：烷烃在光照/高温下与卤素反应，生成混合卤代烷，如：

    CHX3CHX2CHX3+ClX2→hvCHX3CHX2CHX2Cl+(CHX3)X2CHCl\ce{CH3CH2CH3 + Cl2 ->[hv] CH3CH2CH2Cl + (CH3)2CHCl} CHX3​CHX2​CHX3​+ClX2​hv​CHX3​CHX2​CHX2​Cl+(CHX3​)X2​CHCl

  • 芳烃亲电卤化：苯环在 FeXX3\ce{FeX3}FeXX3​ 催化下与卤素反应，生成卤代芳烃，如：

    CX6HX6+BrX2→FeBrX3CX6HX5Br+HBr(126–142∘C)\ce{C6H6 + Br2 ->[FeBr3] C6H5Br + HBr}\quad(126\text{--}142^\circ\mathrm{C}) CX6​HX6​+BrX2​FeBrX3​​CX6​HX5​Br+HBr(126–142∘C)

  • 烯烃烯丙位卤化（烯丙型卤代烃）：高温条件下发生烯丙位取代，如：

    CHX3CH=CHX2+ClX2→500X∘CClCHX2CH=CHX2+HCl\ce{CH3CH=CH2 + Cl2 ->[500^\circ C] ClCH2CH=CH2 + HCl} CHX3​CH=CHX2​+ClX2​500X∘C​ClCHX2​CH=CHX2​+HCl

2. 不饱和烃加成

  • 亲电加成：烯烃/炔烃与 HX\ce{HX}HX、XX2\ce{X2}XX2​ 加成；HX\ce{HX}HX 加成通常符合马氏规则，如：

    CHX3CH=CHX2+HBr→HX2SOX4(CHX3)X2CHBr\ce{CH3CH=CH2 + HBr ->[H2SO4] (CH3)2CHBr} CHX3​CH=CHX2​+HBrHX2​SOX4​​(CHX3​)X2​CHBr

    CHX2=CHX2+BrX2→CClX4BrCHX2CHX2Br\ce{CH2=CH2 + Br2 ->[CCl4] BrCH2CH2Br} CHX2​=CHX2​+BrX2​CClX4​​BrCHX2​CHX2​Br

  • 自由基加成：烯烃与 HBr\ce{HBr}HBr 在过氧化物存在下反马氏加成，如：

    CHX3CH=CHX2+HBr→ROORCHX3CHX2CHX2Br\ce{CH3CH=CH2 + HBr ->[ROOR] CH3CH2CH2Br} CHX3​CH=CHX2​+HBrROOR​CHX3​CHX2​CHX2​Br

3. 由醇制备（最常用）

  • 与氢卤酸（HX\ce{HX}HX）反应：可逆反应，叔醇反应最快，如：

    (CHX3)X3COH+HCl→(CHX3)X3CCl+HX2O\ce{(CH3)3COH + HCl -> (CH3)3CCl + H2O} (CHX3​)X3​COH+HCl​(CHX3​)X3​CCl+HX2​O

    CX6HX11OH+HBr→refluxCX6HX11Br+HX2O\ce{C6H11OH + HBr ->[reflux] C6H11Br + H2O} CX6​HX11​OH+HBrreflux​CX6​HX11​Br+HX2​O

  • 与卤化磷（PClX3\ce{PCl3}PClX3​、PBrX3\ce{PBr3}PBrX3​）反应：避免重排，如：

    3 ROH+PBrX3→3 RBr+HX3POX3\ce{3ROH + PBr3 -> 3RBr + H3PO3} 3ROH+PBrX3​​3RBr+HX3​POX3​

  • 与氯化亚砜（SOClX2\ce{SOCl2}SOClX2​）反应：副产物为气体，产物纯度高，如：

    ROH+SOClX2→RCl+SOX2↑+HCl↑\ce{ROH + SOCl2 -> RCl + SO2 ^ + HCl ^} ROH+SOClX2​​RCl+SOX2​↑+HCl↑

4. 卤原子交换反应

  • 氯代烷/溴代烷与 NaI\ce{NaI}NaI 在丙酮中反应，生成碘代烷（NaCl/NaBr\ce{NaCl/NaBr}NaCl/NaBr 不溶于丙酮），如：

    CHX3CHX2Br+NaI→acetoneCHX3CHX2I+NaBr↓\ce{CH3CH2Br + NaI ->[acetone] CH3CH2I + NaBr v} CHX3​CHX2​Br+NaIacetone​CHX3​CHX2​I+NaBr↓

  • 应用：实验室制备碘代烷，或检验氯代烷/溴代烷。

5. 多卤代烃脱卤化氢

  • 部分脱 HX\ce{HX}HX 生成不饱和卤代烃（示意）：

    BrCHX2CHBrCHX3→NaOH,  HX2O,  ΔCHX2=CBrCHX3+HBr\ce{BrCH2CHBrCH3 ->[NaOH,\ H2O,\ \Delta] CH2=CBrCH3 + HBr} BrCHX2​CHBrCHX3​NaOH, HX2​O, Δ​CHX2​=CBrCHX3​+HBr

  • 邻二卤代烷脱 2 分子 HX\ce{HX}HX 生成炔烃（示意）：

    RCHX2CHXX2→NaNHX2,  NHX3(l)RC≡CH+2 NaX+2 NHX3\ce{RCH2CHX2 ->[NaNH2,\ NH3(l)] RC#CH + 2NaX + 2NH3} RCHX2​CHXX2​NaNHX2​, NHX3​(l)​RC≡CH+2NaX+2NHX3​

6. 其他方法

  • 卤甲基化：芳烃与 (HCHO)X3+HX\ce{(HCHO)3 + HX}(HCHO)X3​+HX 在酸性催化条件下引入 −CHX2X\ce{-CH2X}−CHX2​X（你原式写法与计量不一致，建议只保留“方法描述”，反应式用课本原式替换）
  • 重氮盐（Sandmeyer 反应）：

    ArNX2X+ XX−→CuXArX+NX2↑\ce{ArN2+ X- ->[CuX] ArX + N2 ^} ArNX2​X+ XX−CuX​ArX+NX2​↑

三、卤代烃的物理性质

1. 状态与颜色

  • 氟代烃除外，CHX3Cl\ce{CH3Cl}CHX3​Cl、CHX3Br\ce{CH3Br}CHX3​Br、CHX2=CHCl\ce{CH2=CHCl}CHX2​=CHCl 为气体；其余多为无色液体或固体。
  • 碘代烷/溴代烷长期放置易分解，产生游离碘/溴而显色（碘代烷呈棕色）。

2. 溶解性与气味

  • 不溶于水，溶于乙醇、乙醚、苯、四氯化碳等有机溶剂；部分卤代烃（CHX2ClX2\ce{CH2Cl2}CHX2​ClX2​、CHClX3\ce{CHCl3}CHClX3​）本身是优良溶剂。
  • 多数有不愉快气味；部分卤代烃有毒（如氯乙烯）；苄基型/烯丙型卤代烃常有催泪性。

3. 沸点与相对密度（核心规律）

性质 规律
沸点 碳数增加而升高；同碳数：RI>RBr>RCl>RFRI > RBr > RCl > RFRI>RBr>RCl>RF；异构体：支链越多沸点越低
相对密度 一氯代烷（<1<1<1），一溴代烷/一碘代烷（>1>1>1）；卤代芳烃（>1>1>1）；同系列中随碳数增加而下降

4. 偶极矩

  • C−X\ce{C-X}C−X 键有极性，偶极矩：CHX3Cl\ce{CH3Cl}CHX3​Cl（4.84×10−30 C⋅m4.84\times10^{-30}\ \mathrm{C\cdot m}4.84×10−30 C⋅m）> CHX3Br\ce{CH3Br}CHX3​Br > CHX3I\ce{CH3I}CHX3​I；
  • 乙烯型卤代烃（如 CHX2=CHCl\ce{CH2=CHCl}CHX2​=CHCl）因 ppp–π\piπ 共轭，偶极矩小于相应卤代烷（如 CHX3CHX2Cl\ce{CH3CH2Cl}CHX3​CHX2​Cl）。

（1）亲核性规律

  • 同周期：NHX2X−>OHX−>FX−\ce{NH2^- > OH^- > F^-}NHX2​X−>OHX−>FX−；
  • 同族（极性质子溶剂）：IX−>BrX−>ClX−>FX−\ce{I^- > Br^- > Cl^- > F^-}IX−>BrX−>ClX−>FX−，RSX−>ROX−\ce{RS^- > RO^-}RSX−>ROX−；
  • 同亲核原子：碱性越强一般亲核性越强（溶剂与位阻会改变对应关系）。

（2）卤代烃中 SN2S_{N2}SN2​ 与消除反应（E2E2E2）的竞争规律

在强碱 / 强亲核试剂（如 C2H5ONa/C2H5OHC_2H_5ONa / C_2H_5OHC2​H5​ONa/C2​H5​OH）和较高温度条件下，
卤代烃常发生 SN2S_{N2}SN2​ 亲核取代反应与 E2E2E2 消除反应的竞争。
反应结果主要取决于 α\alphaα-碳和 β\betaβ-碳的空间结构。

1. α\alphaα-碳空间位阻的影响（决定 SN2S_{N2}SN2​ 是否容易）

  • 伯卤代烃：
    α\alphaα-碳空间位阻小，有利于亲核试剂从背面进攻，
    SN2S_{N2}SN2​ 反应容易发生。

  • 仲卤代烃：
    α\alphaα-碳具有一定空间位阻，不利于 SN2S_{N2}SN2​，
    同时具有较多 β\betaβ-氢，
    SN2S_{N2}SN2​ 与 E2E2E2 反应相互竞争。

  • 叔卤代烃：
    α\alphaα-碳空间位阻极大，SN2S_{N2}SN2​ 反应几乎不能发生，
    消除反应占主导。

伯卤代烃  >  仲卤代烃  >  叔卤代烃(SN2 易→E2 易)\text{伯卤代烃} \;>\; \text{仲卤代烃} \;>\; \text{叔卤代烃} \quad (S_{N2}\ \text{易} \rightarrow E2\ \text{易}) 伯卤代烃>仲卤代烃>叔卤代烃(SN2​ 易→E2 易)

2. β\betaβ-氢原子数目的影响（决定消除反应程度）

在强碱条件下，E2E2E2 反应需要从 β\betaβ-碳上夺取 β\betaβ-氢原子：

  • β\betaβ-碳上氢原子数目越多，
    E2E2E2 消除反应越容易发生。

  • 当 β\betaβ-碳上连有支链时：

    • β\betaβ-氢数量增加；
    • 生成的烯烃取代度更高、稳定性更强；
      因而消除产物比例明显增大。

3. 支链效应的双重作用

支链对反应具有两方面促进消除反应的作用：

  1. 增大 α\alphaα-碳的空间位阻，
    不利于 SN2S_{N2}SN2​ 亲核进攻。

  2. 提高 β\betaβ-碳的取代程度，
    有利于生成更稳定的烯烃（符合扎伊采夫规则）。

4. 总结规律（考试版结论）

  • 伯卤代烃：
    SN2S_{N2}SN2​ 为主，消除反应很少。

  • 仲卤代烃：
    SN2S_{N2}SN2​ 与 E2E2E2 并存，反应比例由结构和反应条件决定。

  • 叔卤代烃：
    SN2S_{N2}SN2​ 几乎不发生，E2E2E2 为主。

七、卤代烯烃和卤代芳烃的化学性质

1. 活性顺序（核心：C−X\ce{C-X}C−X 键强度与 ppp–π\piπ 共轭）

烯丙型/苄基型>隔离型>乙烯型/苯基型\boxed{\text{烯丙型/苄基型} > \text{隔离型} > \text{乙烯型/苯基型}} 烯丙型/苄基型>隔离型>乙烯型/苯基型​

  • 乙烯型/苯基型：X\ce{X}X 连 sp2sp^2sp2 碳，ppp–π\piπ 共轭使 C−X\ce{C-X}C−X 键部分双键化（键能高），难发生常规 SN1/SN2S_\mathrm{N}1/S_\mathrm{N}2SN​1/SN​2；
  • 烯丙型/苄基型：共轭可稳定中间体或降低过渡态能量，使取代/消除更易进行。

2. 乙烯型/苯基型卤代烃的特殊反应

（1）亲核取代反应（需强烈条件或吸电子基活化）

  • 加成–消除机理（SNArS_\mathrm{N}ArSN​Ar）：芳环邻/对位连强吸电子基（如 −NOX2\ce{-NO2}−NOX2​、−CN\ce{-CN}−CN）时，亲核试剂先加成生成负离子中间体，再消除 XX−\ce{X^-}XX−。
  • 消除–加成机理（苯炔机理）：无强吸电子基时，强碱条件下先消除形成苯炔，再发生加成。

（2）消除反应

  • 乙烯型卤代烃强烈条件下可脱 HX\ce{HX}HX 生成炔烃（按课本表述掌握条件与机理类型）。

（3）与金属反应

  • 乙烯基/芳基卤代烃在一定条件下也可形成相应金属有机物或参与偶联反应（按课本条目掌握）。

3. 烯丙型/苄基型卤代烃的反应

  • 亲核取代：可走 SN1S_\mathrm{N}1SN​1 或 SN2S_\mathrm{N}2SN​2，并可能发生烯丙基重排；
  • 消除反应：可生成更稳定的烯烃/共轭体系；
  • 与金属反应：较易生成 Grignard 试剂，但也可能出现偶联等副反应（以课本总结为准）。

八、氟代烃（特殊类型）

1. 命名

  • 系统命名：与其他卤代烃一致，全氟化合物前缀“全氟”，如 CX4FX8\ce{C4F8}CX4​FX8​（全氟环丁烷）；

2. 制法

  • 烃的直接氟化（按课本列举条件与特点）；
  • 卤原子交换引入 F\ce{F}F（如 RCl/RBr→RF\ce{RCl/RBr -> RF}RCl/RBr​RF 的方法体系）；

3. 性质与应用

  • 物理性质：沸点低、疏水性强、不溶于水；
  • 化学性质：C−F\ce{C-F}C−F 键能高，多氟/全氟代烃稳定性高，耐酸碱、耐高温；某些一氟代烃在强碱条件下可脱 HF\ce{HF}HF；
  • 应用：曾用作冷冻剂（氟利昂）、阻燃剂、材料（如 PTFE\ce{PTFE}PTFE）。
