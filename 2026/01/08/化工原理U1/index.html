第一章 流体力学基础

本章核心：用静力学与动量/能量守恒描述流体的压力、流动与阻力，并能完成管路计算与流量/流速测量的基本分析。

1.1 概述

1.1.1 化工过程中的流体问题

化工生产中大量涉及：

  • 流体的输送（管路、阀门、泵/风机）
  • 流体的计量与测量（压力、压差、液位、流量）
  • 设备内外的流动阻力与能量消耗
  • 与传热、传质、反应等耦合的流动现象（如边界层、湍流对传递过程的影响）

1.1.2 连续介质模型与“质点”思想

  • 连续介质假设：把流体视作连续分布的介质，用密度、压力、速度等场量描述。
  • 流体微团（质点）：足够小但仍含大量分子的体积元，便于把守恒定律写成工程方程。

1.1.3 流体基本物性（工程常用）

  • 密度：ρ=mV\rho=\dfrac{m}{V}ρ=Vm​
  • 比容：v=1ρv=\dfrac{1}{\rho}v=ρ1​
  • 重度（容重）：γ=ρg\gamma=\rho gγ=ρg
  • 可压缩性
    • 液体通常近似不可压缩（ρ\rhoρ 近似常数）
    • 气体密度随 p,Tp,Tp,T 变化显著

  • 分离会显著增大阻力
  • 主流区：可看成是理想流体，研究流体阻力时，只要集中研究边界层即可。
  • 层流边界层、湍流边界层、进口段长度、层流底层边界层分离发生于流道突然扩大的减速增压过程液体输送时应避免发生边界层分离，而对混合及传热等操作来说应利用这一现象。

  • 阻力显著大于层流
  • 摩擦系数需结合 ReReRe 与粗糙度确定

1.8 因次分析法

1. 基本思想

因次分析法建立在量纲一致性原理之上：
任何正确的物理关系式中，各项的量纲必须一致，与所选单位制无关。

因此，物理过程的函数关系可以用无量纲量之间的关系来表示。

2. 量纲与基本量纲

常用基本量纲包括：

M (质量),L (长度),T (时间),Θ (温度)M\ (\text{质量}),\quad L\ (\text{长度}),\quad T\ (\text{时间}),\quad \Theta\ (\text{温度}) M (质量),L (长度),T (时间),Θ (温度)

任一物理量 qqq 均可表示为：

[q]=MaLbTcΘd[q] = M^{a} L^{b} T^{c} \Theta^{d} [q]=MaLbTcΘd

3. Buckingham Π\PiΠ 定理

若某一物理过程涉及 nnn 个有量纲物理量：

q1,q2,…,qnq_1, q_2, \dots, q_n q1​,q2​,…,qn​

其中包含 kkk 个相互独立的基本量纲，则这些变量之间的关系可化为：

f(Π1,Π2,…,Πn−k)=0f(\Pi_1, \Pi_2, \dots, \Pi_{n-k}) = 0 f(Π1​,Π2​,…,Πn−k​)=0

其中 Πi\Pi_iΠi​ 为无量纲组合量。

4. 无量纲群的构造

选取 kkk 个重复变量（能覆盖全部基本量纲），将其与其余变量组合：

Π=q qr1aqr2b⋯qrkc\Pi = q \, q_{r1}^{a} q_{r2}^{b} \cdots q_{rk}^{c} Π=qqr1a​qr2b​⋯qrkc​

通过令 Π\PiΠ 的总量纲为 111，解指数 a,b,c,…a,b,c,\dotsa,b,c,…，即可得到无量纲群。

5. 因次分析法的工程意义

  • 减少变量数目，简化实验与理论分析
  • 揭示不同物理过程之间的相似性规律
  • 为经验关联式提供理论框架

工程上常将结果写成经验形式：

Π1=C Π2αΠ3β⋯\Pi_1 = C \, \Pi_2^{\alpha} \Pi_3^{\beta} \cdots Π1​=CΠ2α​Π3β​⋯

6. 适用范围与局限

  • 只能确定函数形式结构，不能给出常数 CCC 与指数
  • 需结合实验或理论分析确定具体关系
