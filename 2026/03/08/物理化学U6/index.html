物理化学U6

第六章 相平衡

一、相律基础

1. 核心概念定义

（1）相（PPP）

  • 系统中物理性质和化学性质完全均匀一致的部分，相与相间存在明显界面，穿过界面性质发生突变；
  • 相数：系统中相的总数，用PPP表示：
    • 气体：无论含多少种气体，均为单相（混合气体分子均匀分散）；
    • 液体：按互溶程度分单相（如水+乙醇）、二相（如水+苯）等，通常不超过三相；
    • 固体：除固溶体（原子/离子均匀混合）为单相外，有几种独立固体物质即至少几种固相；同物质不同晶体结构为不同相（如石墨与金刚石、白磷与红磷）。

（2）物种数（SSS）

  • 定义：系统中所有化学物质的种类数，用SSS表示；
  • 判断时只看化学物质种类，相态不同不额外增加物种数。

（3）组分（CCC）

  • 定义：能独立构成系统所有相的最少物质种类数，用CCC表示；
  • 按理解可看作：描述体系组成所需的最少独立参数数目；
  • 计算关系：C=S−R−R′C = S - R - R'C=S−R−R′
    • SSS：系统中所有物质的种类数（物种数）；
    • RRR：系统中独立的化学平衡关系式数目；
    • R′R'R′：系统中独立的浓度限制条件数目（仅适用于同一相内的浓度关系）。

（4）自由度（FFF）

  • 定义：在不引起旧相消失、新相产生的前提下，可独立改变的强度变量（温度、压力、浓度等）数目，用FFF表示；
  • 按理解可看作：在已知体系组成后，描述体系状态还需要的独立状态参数数目。

2. 个人理解提醒：SSS、CCC、FFF

（1）SSS

  • SSS表示体系里有多少种物质；
  • 只看物质种类，不看相态。

（2）CCC

  • CCC表示描述体系组成需要多少个独立物种；
  • 原本若要描述SSS个物种的组成，需要SSS个信息；
  • 但由于存在：
    • RRR个独立反应关系；
    • R′R'R′个独立浓度限制关系；
  • 所以独立描述量减少，得到：

C=S−R−R′C = S - R - R' C=S−R−R′

（3）FFF

  • FFF表示在不改变相数的条件下，体系状态还能自由选择多少个参数；
  • 可理解为：描述体系状态时，除了组成之外，还剩多少个独立强度变量可选。

3. 强度变量说明

  • 强度变量：与体系大小无关的变量；
  • 本章相律中主要涉及：
    • 温度TTT
    • 压力ppp
    • 浓度
    • 摩尔分数
  • 不包括体积、质量、物质的量等广延变量。

4. 相律表达式

（1）吉布斯相律核心公式

F=C−P+2F = C - P + 2 F=C−P+2

  • 推导依据：热力学平衡条件（各相温度、压力相等，各物质化学势在各相间相等）；
  • 式中“2”：代表温度、压力两个独立的外界强度变量（适用于无外场影响的系统）。

（2）条件自由度（F′F'F′）

  • 定义：指定某一/几个强度变量（如定温、定压）后，剩余可独立改变的强度变量数目；
  • 常见形式：
    • 定温条件（TTT固定）：F′=C−P+1F' = C - P + 1F′=C−P+1；
    • 定压条件（ppp固定）：F′=C−P+1F' = C - P + 1F′=C−P+1；
    • 定温定压条件（T、pT、pT、p均固定）：F′=C−PF' = C - PF′=C−P。

（3）相律应用限制

  • 仅适用于热力学平衡系统；
  • 不考虑电场、磁场、重力场等外场影响（若考虑，公式中“2”需替换为对应外场变量数）；
  • 强度变量仅涉及温度、压力、浓度，不包括体积、质量等广延变量。

5. 关于化学势条件的个人提醒

（1）正确理解

  • 相平衡时，同一物种在各相中的化学势相等：

μiα=μiβ\mu_i^\alpha = \mu_i^\beta μiα​=μiβ​

  • 这个关系是相平衡条件，不是任意状态下都成立。

（2）容易产生的误解

  • 误以为：任一物种在任意状态下都满足

μiα=μiβ\mu_i^\alpha = \mu_i^\beta μiα​=μiβ​

  • 实际上只有在两相或多相处于平衡时才成立；
  • 若

μiα≠μiβ\mu_i^\alpha \ne \mu_i^\beta μiα​=μiβ​

则体系会发生相变，直到达到新的平衡。

二、单组分系统的相平衡

1. 单组分系统的相律分析

  • 单组分系统：C=1C = 1C=1，代入相律公式得：F=1−P+2=3−PF = 1 - P + 2 = 3 - PF=1−P+2=3−P
    • P=1P=1P=1（单相系统）：F=2F=2F=2（双变量系统，T、pT、pT、p可独立改变）；
    • P=2P=2P=2（两相平衡）：F=1F=1F=1（单变量系统，T、pT、pT、p只有一个可独立改变）；
    • P=3P=3P=3（三相平衡）：F=0F=0F=0（无变量系统，T、pT、pT、p均固定，为三相点）；
    • 结论：单组分系统最多只能有三相共存（F≥0F\geq0F≥0，故P≤3P\leq3P≤3）。

2. 单组分系统的相图（p−Tp-Tp−T图）

（1）相图基本特征

  • 坐标：以压力ppp为纵轴，温度TTT为横轴；
  • 曲线：两相平衡线（F=1F=1F=1），描述两相平衡时ppp与TTT的关系；
  • 区域：单相区（F=2F=2F=2），区域内T、pT、pT、p可自由改变，相态不变；
  • 点：三相点（F=0F=0F=0），三条两相平衡线的交点，对应唯一的T、pT、pT、p组合。

（2）两相平衡线的热力学方程——克拉佩龙方程

① 公式推导与表达式

  • 推导基础：两相平衡时化学势相等（μα(T,p)=μβ(T,p)\mu^\alpha(T,p) = \mu^\beta(T,p)μα(T,p)=μβ(T,p)），结合吉布斯自由能微分式推导；
  • 微分形式：

dpdT=ΔHmTΔVm\frac{dp}{dT} = \frac{\Delta H_m}{T\Delta V_m} dTdp​=TΔVm​ΔHm​​

  • ΔHm\Delta H_mΔHm​：相变摩尔焓（如蒸发热ΔvapHm\Delta_{vap}H_mΔvap​Hm​、熔化热ΔfusHm\Delta_{fus}H_mΔfus​Hm​、升华热ΔsubHm\Delta_{sub}H_mΔsub​Hm​）；
  • ΔVm\Delta V_mΔVm​：相变摩尔体积（ΔVm=Vmβ−Vmα\Delta V_m = V_m^\beta - V_m^\alphaΔVm​=Vmβ​−Vmα​）；
  • TTT：相变温度（热力学温度）。

② 克拉佩龙方程的应用

  • 适用于任何单组分系统的两相平衡（气-液、气-固、液-固平衡）；
  • 预测温度对平衡压力的影响：
    • 熔化过程（液-固平衡）：多数物质ΔfusHm>0\Delta_{fus}H_m>0Δfus​Hm​>0，ΔVm>0\Delta V_m>0ΔVm​>0（冰除外），故dpdT>0\frac{dp}{dT}>0dTdp​>0；
    • 蒸发过程（气-液平衡）：ΔvapHm>0\Delta_{vap}H_m>0Δvap​Hm​>0，ΔVm>0\Delta V_m>0ΔVm​>0，故dpdT>0\frac{dp}{dT}>0dTdp​>0；
    • 升华过程（气-固平衡）：ΔsubHm>0\Delta_{sub}H_m>0Δsub​Hm​>0，ΔVm>0\Delta V_m>0ΔVm​>0，故dpdT>0\frac{dp}{dT}>0dTdp​>0。

（3）克劳修斯-克拉佩龙方程（气-液/气-固平衡近似式）

① 适用条件与简化依据

  • 适用于气-液平衡或气-固平衡；
  • 简化假设：
    • 气相视为理想气体，Vm(g)≫Vm(l)V_m(g) \gg V_m(l)Vm​(g)≫Vm​(l)（或Vm(s)V_m(s)Vm​(s)），故ΔVm≈Vm(g)=RTp\Delta V_m \approx V_m(g) = \frac{RT}{p}ΔVm​≈Vm​(g)=pRT​；
    • 相变摩尔焓ΔHm\Delta H_mΔHm​在指定温度范围内视为常数。

② 方程形式

  • 微分形式：

dln⁡pdT=ΔHmRT2\frac{d\ln p}{dT} = \frac{\Delta H_m}{RT^2} dTdlnp​=RT2ΔHm​​

  • 定积分形式（常用）：

ln⁡p=−ΔHmRT+C\ln p = -\frac{\Delta H_m}{RT} + C lnp=−RTΔHm​​+C

ln⁡p2p1=ΔHmR(1T1−1T2)\ln\frac{p_2}{p_1} = \frac{\Delta H_m}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right) lnp1​p2​​=RΔHm​​(T1​1​−T2​1​)

  • 适用场景：计算不同压力下的相变温度（如沸点、升华点），或已知两温度下的平衡压力求相变摩尔焓。

（4）单组分系统相图示例（以水为例）

  • 三相点：T=273.16KT=273.16KT=273.16K（0.01∘C0.01^\circ C0.01∘C），p=610.62Pap=610.62Pap=610.62Pa，为气、液、固三相共存点；
  • 三条平衡线：
    • 蒸气压线（气-液线）：从三相点延伸至临界点；
    • 升华线（气-固线）：从三相点向左下方延伸；
    • 熔化线（液-固线）：从三相点向左上方延伸，斜率为负（因冰的摩尔体积大于水，ΔVm<0\Delta V_m<0ΔVm​<0），故压力升高，冰的熔点降低。

3. 外压对蒸气压的影响

  • 在恒温条件下，分析纯液体与其蒸气平衡体系可得：外压增大时，蒸气压也增大；
  • 关系式表明：

dpgdpe>0\frac{d p_g}{d p_e} > 0 dpe​dpg​​>0

  • 但由于气相摩尔体积远大于液相摩尔体积，因此一般条件下外压对蒸气压影响很小；
  • 在接近临界状态时，此影响会变得明显。

4. 工程经验方程

  • 对于液体饱和蒸气压与温度关系，工程上常采用安托因方程：

lg⁡p=A−Bt+C\lg p = A - \frac{B}{t+C} lgp=A−t+CB​

  • 其中：
    • A,B,CA,B,CA,B,C为经验常数；
    • ttt为摄氏温度。
