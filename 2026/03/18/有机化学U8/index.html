有机化学U8

第八章 波谱分析基础

一、总述：本章研究什么

本章核心不是“化学反应”，而是利用波谱信息推断有机化合物结构。
主要包括两条主线：

  1. 红外光谱（IR）：判断分子中有哪些官能团；
  2. 核磁共振谱（NMR）：判断分子中不同类型原子的化学环境。

因此，本章的任务可以概括为：

分子结构⟶波谱特征⟶结构鉴定\text{分子结构} \longrightarrow \text{波谱特征} \longrightarrow \text{结构鉴定} 分子结构⟶波谱特征⟶结构鉴定

二、分子吸收光谱与分子结构

1. 吸收光谱的本质

分子只有在吸收的辐射能量恰好等于能级差时，才会发生跃迁并产生吸收峰。

E=hνE = h \nu E=hν

其中：

  • EEE：能量差
  • hhh：Planck 常数
  • ν\nuν：频率

频率、波长与波数的关系为：

ν=cλ=cσ\nu = \frac{c}{\lambda} = c \sigma ν=λc​=cσ

其中：

  • λ\lambdaλ：波长
  • σ\sigmaσ：波数，常用单位为 cm−1\mathrm{cm}^{-1}cm−1

2. 不同波段对应的分子运动

不同波段的电磁波会引起不同类型的分子运动：

  • 紫外—可见光：电子跃迁
  • 红外光：分子振动与转动
  • 无线电波区：核磁共振

所以在有机化学中：

  • 红外光谱主要用于分析官能团；
  • 核磁共振谱主要用于分析原子所处环境与分子骨架。

三、红外光谱（IR）

3.1 红外吸收的基本原理

1. 振动频率

化学键可近似看作弹簧连接的两个原子，其振动频率满足：

ν=12πk(1m1+1m2)\nu = \frac{1}{2\pi}\sqrt{k\left(\frac{1}{m_1}+\frac{1}{m_2}\right)} ν=2π1​k(m1​1​+m2​1​)​

更常写成与波数相关的形式。其定性结论是：

  • 键力常数越大，吸收频率越高；
  • 原子质量越小，吸收频率越高。

因此一般有：

  • O—H、N—H 吸收频率较高；
  • C=C、C=O 高于 C—C、C—O；
  • 三键区通常高于双键区。

2. 振动类型

分子振动主要分为两类：

（1）伸缩振动

键长发生变化，键角基本不变。

（2）弯曲振动

键角发生变化，键长变化较小。

常见弯曲振动包括：

  • 剪式振动
  • 面内摇摆振动
  • 面外摇摆振动
  • 卷曲振动

本章图 8-1 给出了甲叉基的几种典型振动方式。

3.2 特征频率与特征吸收区

1. 特征频率

同类官能团通常在固定波数范围内出现吸收峰，这一范围称为该官能团的特征频率。

常见重要区间如下：

（1）O—H

3600∼3200 cm−13600 \sim 3200\ \mathrm{cm}^{-1} 3600∼3200 cm−1

特点：

  • 醇、酚的 O—H 常为宽峰；
  • 氢键越强，峰越宽，且峰位常向低波数移动。

（2）N—H

3500∼3300 cm−13500 \sim 3300\ \mathrm{cm}^{-1} 3500∼3300 cm−1

特点：

  • 伯胺常出现两个峰；
  • 仲胺常出现一个峰；
  • 叔胺无 N—H 伸缩峰。

（3）C—H

不同杂化状态差别明显：

  • sp3sp^3sp3 C—H：

    3000∼2850 cm−13000 \sim 2850\ \mathrm{cm}^{-1} 3000∼2850 cm−1

  • sp2sp^2sp2 C—H：

    3100∼3000 cm−13100 \sim 3000\ \mathrm{cm}^{-1} 3100∼3000 cm−1

  • spspsp C—H：

    3300 cm−1附近3300\ \mathrm{cm}^{-1}\text{附近} 3300 cm−1附近

所以：

  • 烯烃、芳烃的 =C—H 往往略高于 3000 cm−13000\ \mathrm{cm}^{-1}3000 cm−1；
  • 烷烃的 C—H 一般低于 3000 cm−13000\ \mathrm{cm}^{-1}3000 cm−1。

（4）C=C、C=O、C≡N、C≡C

这是判断不饱和键和羰基的关键区域：

  • C=C：

    1680∼1620 cm−11680 \sim 1620\ \mathrm{cm}^{-1} 1680∼1620 cm−1

  • C=O：

    1850∼1650 cm−11850 \sim 1650\ \mathrm{cm}^{-1} 1850∼1650 cm−1

    是最强、最重要的特征峰之一
  • C≡N：

    2260∼2220 cm−12260 \sim 2220\ \mathrm{cm}^{-1} 2260∼2220 cm−1

  • C≡C：

    2260∼2100 cm−12260 \sim 2100\ \mathrm{cm}^{-1} 2260∼2100 cm−1

2. 指纹区

通常把

1300∼400 cm−11300 \sim 400\ \mathrm{cm}^{-1} 1300∼400 cm−1

称为指纹区。

特点：

  • 吸收峰多而复杂；
  • 对同分异构体非常敏感；
  • 常用于与标准谱图对照确认结构。

而

4000∼1300 cm−14000 \sim 1300\ \mathrm{cm}^{-1} 4000∼1300 cm−1

常称为特征频率区，更适合先做官能团判断。
这些划分和表 8-2 在本章中写得很清楚。

3.3 红外图谱判读要点

1. 烷烃

  • sp3sp^3sp3 C—H 伸缩振动：

    2960∼2860 cm−12960 \sim 2860\ \mathrm{cm}^{-1} 2960∼2860 cm−1

  • 甲基、亚甲基弯曲振动常见于：

    1467, 1380 cm−11467,\ 1380\ \mathrm{cm}^{-1} 1467, 1380 cm−1

  • 长链 (CH2)n(CH_2)_n(CH2​)n​ 常在约

    721 cm−1721\ \mathrm{cm}^{-1} 721 cm−1

    有峰。

2. 烯烃

  • =C—H 伸缩：

    3080 cm−1附近3080\ \mathrm{cm}^{-1}\text{附近} 3080 cm−1附近

  • C=C 伸缩：

    1642 cm−1附近1642\ \mathrm{cm}^{-1}\text{附近} 1642 cm−1附近

  • 末端双键常有面外弯曲吸收，可辅助判断取代类型。

3. 炔烃

  • 末端炔 =C—H：

    3300 cm−1附近3300\ \mathrm{cm}^{-1}\text{附近} 3300 cm−1附近

  • C≡C：

    2100 cm−1附近2100\ \mathrm{cm}^{-1}\text{附近} 2100 cm−1附近

4. 芳烃

  • 芳环 C—H：

    3100∼3000 cm−13100 \sim 3000\ \mathrm{cm}^{-1} 3100∼3000 cm−1

  • 苯环骨架振动：

    1600∼1450 cm−11600 \sim 1450\ \mathrm{cm}^{-1} 1600∼1450 cm−1

  • 单取代苯、邻二取代、间二取代、对二取代常在面外弯曲区表现不同，可用于判别取代型。

5. 卤代烃

  • C—X 键通常出现在低波数区；
  • 常用于辅助判断，不作为最强结构依据。

四、核磁共振氢谱（1H^1\mathrm{H}1H NMR）

4.1 核磁共振的产生

1. 核自旋与能级分裂

某些原子核（如 1H^1\mathrm{H}1H）具有自旋。放在外磁场 B0B_0B0​ 中时，会分裂成不同能级。
对于氢核，只存在两个取向，对应两个能级。

当满足共振条件时，原子核吸收无线电波并发生跃迁。

2. 共振条件

其本质可写成：

ΔE=hν\Delta E = h \nu ΔE=hν

外加磁场越强，共振频率越高。

4.2 1H^1\mathrm{H}1H NMR 图谱中的四个核心信息

判断氢谱，最重要的就是四件事：

  1. 峰的个数
  2. 化学位移 δ\deltaδ
  3. 峰面积（积分）
  4. 裂分情况

4.3 化学位移

1. 定义

通常以四甲基硅烷 TMS 为标准，定义化学位移：

δ=ν样−νTMSν0×106\delta = \frac{\nu_{\text{样}}-\nu_{\text{TMS}}}{\nu_0}\times 10^6 δ=ν0​ν样​−νTMS​​×106

单位是 ppm。

2. 本质

不同氢核周围电子云密度不同，屏蔽作用不同，因此共振位置不同。

  • 电子云密度大 → 屏蔽强 → δ\deltaδ 小 → 上场
  • 电子云密度小 → 去屏蔽 → δ\deltaδ 大 → 下场

3. 常见化学位移范围

本章表 8-3 给出了常见氢的化学位移，可整理为：

  • 烷基氢：

    0.8∼1.50.8 \sim 1.5 0.8∼1.5

  • 与 π\piπ 键或杂原子相邻的烷基氢：

    2∼4.52 \sim 4.5 2∼4.5

  • 烯烃氢：

    4.6∼6.54.6 \sim 6.5 4.6∼6.5

  • 芳香氢：

    6.5∼8.56.5 \sim 8.5 6.5∼8.5

  • 醛基氢：

    9.5∼109.5 \sim 10 9.5∼10

  • 羧酸氢：

    10∼1310 \sim 13 10∼13

  • 醇、酚、胺上的活泼氢位置变化较大，常不太固定。

这些范围和书上的表 8-3 一致。

4.4 影响化学位移的因素

1. 诱导效应

与电负性较大的原子相连时，氢核周围电子云减少，化学位移增大。

例如同系变化中：

CH4<CH3I<CH3Br<CH3Cl<CH3OH<CH3F\mathrm{CH_4 < CH_3I < CH_3Br < CH_3Cl < CH_3OH < CH_3F} CH4​<CH3​I<CH3​Br<CH3​Cl<CH3​OH<CH3​F

电负性越大，去屏蔽越明显。

2. 磁各向异性

π\piπ 键电子在外磁场中会形成附加磁场，使某些区域氢核受屏蔽，某些区域去屏蔽。

典型表现：

  • 烯烃氢通常比普通烷基氢更靠下场；
  • 醛基氢由于 C=O 的磁各向异性，显著下场；
  • 炔氢反而相对没那么靠下场；
  • 芳香环电流使芳氢通常出现在

    6.5∼8.56.5 \sim 8.5 6.5∼8.5

    区域。

本章图 8-12 到图 8-15 专门说明了 C=C、C=O、C≡C、苯环的磁各向异性。

3. 氢键缔合

O—H、N—H 等活泼氢受氢键影响明显，峰位常漂移且较宽。

4.5 自旋偶合与峰裂分

1. 裂分的原因

若某氢核邻近的氢核不等价，则它们之间通过键的磁相互作用会引起信号裂分。

2. n+1n+1n+1 规律

若某组氢与相邻碳上 nnn 个等价氢偶合，则通常裂分为n+1n+1n+1重峰。

典型例子：

  • 邻近 1 个氢：二重峰
  • 邻近 2 个氢：三重峰
  • 邻近 3 个氢：四重峰

3. 裂分强度比

  • 二重峰：1:11:11:1
  • 三重峰：1:2:11:2:11:2:1
  • 四重峰：1:3:3:11:3:3:11:3:3:1

4. 裂分判断规则

判断裂分时常用：

  1. 先看是否存在邻氢；
  2. 一般只考虑相邻碳上的氢；
  3. 等价氢之间通常不裂分；
  4. 活泼氢（如 O—H）常不明显参与裂分。

4.6 积分曲线

峰面积与该类氢的数目成正比。
因此积分值可以帮助判断：

1H, 2H, 3H, 6H 等1H,\ 2H,\ 3H,\ 6H \text{ 等} 1H, 2H, 3H, 6H 等

例如乙醇的氢谱常能看到三组峰，其积分比对应为：

3:2:13:2:1 3:2:1

本章图 8-11 正是这一例子。

五、13C^{13}\mathrm{C}13C NMR 简介

1. 特点

13C^{13}\mathrm{C}13C 天然丰度低，因此灵敏度低于 1H^1\mathrm{H}1H NMR，但对判断碳骨架非常有用。

2. 化学位移范围更宽

通常大致位于：

0∼2200 \sim 220 0∼220

ppm。

3. 常见区间

可粗略记忆为：

  • 饱和烷基碳：

    0∼500 \sim 50 0∼50

  • 与 O、N、X 相连的饱和碳：

    50∼9050 \sim 90 50∼90

  • 烯烃、芳烃碳：

    100∼160100 \sim 160 100∼160

  • 羰基碳：

    160∼220160 \sim 220 160∼220

六、IR 与 NMR 联合解析思路

这一章最后真正重要的，不是背每一个数字，而是学会联合判断。

6.1 一般步骤

第一步：先看分子式

计算不饱和度：

Ω=2C+2+N−H−X2\Omega = \frac{2C+2+N-H-X}{2} Ω=22C+2+N−H−X​

用于判断是否有：

  • 双键
  • 三键
  • 环
  • 芳环

第二步：看 IR

先找强特征峰：

  • 有无 O—H / N—H
  • 有无 C=O
  • 有无 C=C / C≡C / C≡N
  • 是否像芳环

第三步：看 1H^1\mathrm{H}1H NMR

重点看：

  • 峰数：有几种氢
  • 位移：这些氢大概在哪类环境
  • 积分：每组有几个氢
  • 裂分：相邻有几个氢

第四步：必要时结合 13C^{13}\mathrm{C}13C NMR

看：

  • 有几种碳
  • 是否有羰基碳
  • 是否有芳香 / 烯烃碳
  • 是否存在对称性

6.2 识谱中的常见对应关系

（1）若 IR 有强峰在

1700 cm−1附近1700\ \mathrm{cm}^{-1}\text{附近} 1700 cm−1附近

则优先考虑有 C=O。

（2）若 1H^1\mathrm{H}1H NMR 在

9.5∼109.5 \sim 10 9.5∼10

有峰，则优先考虑 醛基氢。

（3）若 1H^1\mathrm{H}1H NMR 在

6.5∼8.56.5 \sim 8.5 6.5∼8.5

出现多组峰，则常提示 芳环氢。

（4）若有积分为 3 的三重峰 + 积分为 2 的四重峰，

则首先想到：

−CH2CH3-CH_2CH_3 −CH2​CH3​

（5）若某峰很宽、位置不固定，且可交换消失，

则常考虑：

  • O—H
  • N—H

七、本章最应该掌握的结论

1. IR 的作用

先判断官能团

2. 1H^1\mathrm{H}1H NMR 的作用

判断氢环境、数目及连接关系

3. 13C^{13}\mathrm{C}13C NMR 的作用

辅助确定碳骨架与对称性

4. 真正的解题方法

分子式 + IR + 1H^1\mathrm{H}1H NMR + 13C^{13}\mathrm{C}13C NMR 联合推断结构

八、易错点

1.

IR 不能单独精确写出结构，只能先判断官能团。

2.

化学位移不是死记单个数值，而是记范围。

3.

活泼氢（O—H、N—H）峰位不稳定，裂分也常不典型。

4.

峰裂分只对“邻近不等价氢”明显有效，等价氢之间一般不裂分。

5.

芳香环、双键、三键对化学位移的影响，不能只看诱导效应，还要考虑磁各向异性。

本章的本质是：
