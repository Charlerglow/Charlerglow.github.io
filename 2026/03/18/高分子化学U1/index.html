高分子化学U1

第一章 绪论

1.1 高分子基本概念

1. 高分子化合物定义

高分子化合物是由结构单元通过共价键重复连接形成的大分子量同系混合物。

主要特征：

  • 由结构单元重复组成
  • 以共价键连接
  • 分子量大（通常大于 10410^4104）
  • 属于同系混合物

2. 单体、结构单元、重复单元、单体单元

（1）单体（Monomer）

能够发生聚合反应形成聚合物的小分子。

（2）结构单元（Structural Unit）

单体通过聚合反应进入高分子链后的基本结构单元。

（3）重复单元（Repeat Unit）

在聚合物链中化学组成和结构完全重复的最小单元。

（4）单体单元（Monomer Unit）

元素组成与单体相同的结构单元。

四者关系：

  • 均聚物（单一种类单体）（聚合）：

重复单元=结构单元=单体单元\text{重复单元} = \text{结构单元} = \text{单体单元} 重复单元=结构单元=单体单元

  • 均聚物其他情况（缩聚）：

重复单元=结构单元≠单体单元\text{重复单元} = \text{结构单元} \ne \text{单体单元} 重复单元=结构单元=单体单元

  • 共聚物（多种类单体）：

重复单元≠结构单元≠单体单元\text{重复单元} \ne \text{结构单元} \ne \text{单体单元} 重复单元=结构单元=单体单元

例：尼龙-66（缩聚反应）

单体：

  • 己二胺：H2N−(CH2)6−NH2H_2N-(CH_2)_6-NH_2H2​N−(CH2​)6​−NH2​
  • 己二酸：HOOC−(CH2)4−COOHHOOC-(CH_2)_4-COOHHOOC−(CH2​)4​−COOH

聚合后结构片段：

−NH−(CH2)6−NH−CO−(CH2)4−CO−- NH-(CH_2)_6-NH-CO-(CH_2)_4-CO - −NH−(CH2​)6​−NH−CO−(CH2​)4​−CO−

分析：

  • 结构单元：

−NH−(CH2)6−NH−和−CO−(CH2)4−CO−- NH-(CH_2)_6-NH - \quad 和 \quad - CO-(CH_2)_4-CO - −NH−(CH2​)6​−NH−和−CO−(CH2​)4​−CO−

（分别来源于两个不同单体）

  • 重复单元：

−NH−(CH2)6−NH−CO−(CH2)4−CO−- NH-(CH_2)_6-NH-CO-(CH_2)_4-CO - −NH−(CH2​)6​−NH−CO−(CH2​)4​−CO−

（完整重复出现的最小结构）

  • 单体单元：

分别对应两个单体的组成部分，但在聚合过程中发生了脱水反应，因此：

单体单元≠结构单元\text{单体单元} \ne \text{结构单元} 单体单元=结构单元

结论：

该体系属于：

重复单元≠结构单元≠单体单元\text{重复单元} \ne \text{结构单元} \ne \text{单体单元} 重复单元=结构单元=单体单元

3. 聚合度

（1）基于重复单元：

DP=nDP = n DP=n

表示重复单元的个数。

（2）基于结构单元（数均聚合度）：

Xˉn=初始单体分子数生成大分子数\bar{X}_n = \frac{\text{初始单体分子数}}{\text{生成大分子数}} Xˉn​=生成大分子数初始单体分子数​

当一个重复单元包含多个结构单元时：

Xˉn=k⋅n\bar{X}_n = k \cdot n Xˉn​=k⋅n

其中 kkk 为每个重复单元所包含的结构单元数。

聚合度与分子量关系：

M=n⋅M0M = n \cdot M_0 M=n⋅M0​

其中 M0M_0M0​ 为重复单元分子量。

1.2 聚合物的分类和命名

1. 分类

按来源：

  • 天然高分子
  • 合成高分子
  • 改性高分子

按用途：

  • 塑料
  • 橡胶
  • 纤维
  • 涂料
  • 胶粘剂
  • 功能高分子

按热行为：

  • 热塑性聚合物
  • 热固性聚合物

按凝聚态：

  • 玻璃态
  • 橡胶态
  • 部分结晶态

按主链结构：

  • 碳链聚合物
  • 杂链聚合物
  • 元素有机聚合物

2. 命名

（1）习惯命名法

按单体来源：

“聚” + 单体名称\text{“聚” + 单体名称} “聚” + 单体名称

按结构特征命名：用于结构发生变化或单体不唯一情况。

由两种单体合成的共聚物，常摘取两单体的简名，后缀“树脂”两字来命名，例如苯酚和甲醛的缩聚物称为酚醛树脂。合成橡胶往往从共聚单体中各取一字，后缀“橡胶”二字来命名，如丁（二烯）苯（乙烯）橡胶、乙（烯）丙（烯）橡胶等。

商品名：树脂、橡胶、纶等。

英文缩写：如 PEG、ABS、EVA、PC、PTFE。

（2）结构系统命名法（IUPAC）

步骤：

  1. 确定最小重复单元
  2. 排列次级单元顺序
  3. 按有机命名规则命名
  4. 前加“聚”

1.3 聚合反应

1. 定义

由低分子单体生成聚合物的反应称为聚合反应。

2. 按结构变化分类

（1）加聚反应

  • 单体含不饱和键
  • 无副产物
  • 结构单元与单体组成相同
  • 分子量为单体分子量的整数倍

（2）缩聚反应

  • 官能团之间反应
  • 有低分子副产物
  • 分子量不为整数倍关系
  • 多为杂链聚合物

（3）开环聚合

  • 环状单体开环形成聚合物
  • 无副产物
  • 结构类似缩聚物

3. 按机理分类

（1）逐步聚合

  • 无活性中心
  • 官能团逐步反应
  • 分子量逐渐增加
  • 存在中间分子量产物
  • 每步速率和活化能大致相同

（2）连锁聚合

  • 存在活性中心

类型：

  • 自由基聚合
  • 阳离子聚合
  • 阴离子聚合

特点：

  • 包括引发、增长、终止过程
  • 分子量迅速形成

1.4 分子量及其分布

1. 分子量特点

（1）分子量大

影响材料的力学性能和加工性能。

（2）存在分布

聚合物由不同分子量分子组成，分子量为统计平均值。

1. 多层次结构

  • 一级结构：序列结构
  • 二级结构：局部构象
