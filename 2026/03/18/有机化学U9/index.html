第九章 醇和酚

一、醇和酚的分类与命名

1. 分类

（1）按羟基数目分类

  • 一元醇、一元酚
  • 二元醇、二元酚
  • 三元及多元醇、多元酚

例：

一元醇

CH3OHCH_3OH CH3​OH

二元醇

HOCH2CH2OHHOCH_2CH_2OH HOCH2​CH2​OH

三元醇

HOCH2CHOHCH2OHHOCH_2CHOHCH_2OH HOCH2​CHOHCH2​OH

（2）按烃基分类

  • 脂肪醇
  • 脂环醇
  • 芳香醇

注意区分：

  • 芳香醇：Ph−CH2OHPh-CH_2OHPh−CH2​OH
  • 酚：Ph−OHPh-OHPh−OH

（3）按碳类型分类

  • 伯（1∘1^\circ1∘）醇
  • 仲（2∘2^\circ2∘）醇
  • 叔（3∘3^\circ3∘）醇

2. 命名

（1）普通命名法

  • 甲醇、乙醇
  • 异丙醇
  • 苄醇
  • 邻甲酚

（2）系统命名法

原则：

  • 选最长碳链
  • 羟基编号最小

例：

HC≡C−CH(OH)CH2CH3HC \equiv C-CH(OH)CH_2CH_3 HC≡C−CH(OH)CH2​CH3​

→ 戊-4-炔-2-醇

Ph−CH2CH2CH2OHPh-CH_2CH_2CH_2OH Ph−CH2​CH2​CH2​OH

→ 3-苯基丙-1-醇

（3）酚命名

以酚为母体：

  • 2-甲基苯酚
  • 2-异丙基-5-甲基苯酚

二、醇和酚的结构

1. 醇的结构

  • 氧原子为 sp3sp^3sp3 杂化
  • 分子呈近似四面体结构
  • 氧原子上有两对孤对电子

因此：

  • 分子具有极性
  • 分子间可形成氢键

2. 酚的结构

  • 羟基直接与芳环相连
  • 氧原子孤对电子可与苯环发生 p−πp-\pip−π 共轭

因此：

  • C−OC-OC−O 键具有部分双键性
  • 分子更加稳定

3. 结构差异本质

醇：R-OH
酚：Ph-OH

关键区别：

  • 醇：无共轭
  • 酚：存在共轭

三、醇和酚的制法

3.1 醇的制法

（1）工业合成

（2）烯烃水合

C=C+H2O→ROHC=C + H_2O \rightarrow ROH C=C+H2​O→ROH

（3）卤代烃水解

R−X+NaOH→R−OH+NaXR-X + NaOH \rightarrow R-OH + NaX R−X+NaOH→R−OH+NaX

（4）Grignard

RMgX+C=O→ROHRMgX + C=O \rightarrow ROH RMgX+C=O→ROH

（5）还原

  • 醛 → 一级醇
  • 酮 → 二级醇

3.2 酚的制法

（1）异丙苯法

工业上制苯酚的主要方法。异丙苯先被空气氧化为氢过氧化异丙苯，再在酸作用下裂解，生成苯酚和丙酮。

（2）苯磺酸钠的碱熔

苯磺酸或其钠盐在强碱熔融条件下可转化为酚盐，酸化后得到酚。

ArSO3Na+2NaOH→ΔArONa+Na2SO3+H2O\mathrm{ArSO_3Na} + 2\mathrm{NaOH} \xrightarrow{\Delta} \mathrm{ArONa} + \mathrm{Na_2SO_3} + \mathrm{H_2O} ArSO3​Na+2NaOHΔ​ArONa+Na2​SO3​+H2​O

ArONa+H+→ArOH\mathrm{ArONa} + \mathrm{H^+} \rightarrow \mathrm{ArOH} ArONa+H+→ArOH

（3）卤代芳烃的水解

卤原子的邻位或对位连有强吸电子基时，水解反应较易进行，工业上制对硝基苯酚、对氯苯酚等常用此法。

（4）重氮盐的水解

芳香重氮盐水解可制酚，是实验室和有机合成中常用方法。

ArN2+HSO4−+H2O→50∘CArOH+N2↑+H2SO4\mathrm{ArN_2^+HSO_4^-} + \mathrm{H_2O} \xrightarrow{50^\circ C} \mathrm{ArOH} + \mathrm{N_2}\uparrow + \mathrm{H_2SO_4} ArN2+​HSO4−​+H2​O50∘C​ArOH+N2​↑+H2​SO4​

五、醇的化学性质

5.1 酸碱性

（1）醇的酸性

  • 可与活泼金属反应
  • 一般不能与 NaOH 反应

CH3OH>1∘>2∘>3∘CH_3OH > 1^\circ > 2^\circ > 3^\circ CH3​OH>1∘>2∘>3∘

PhO− 共振稳定PhO^- \text{ 共振稳定} PhO− 共振稳定

5.3 成醚反应

Williamson 合成法

醇失去质子后形成的烷氧负离子是强亲核试剂，可与适当的卤代烃发生亲核取代生成醚：

RO−+R′X→ROR′+X−RO^- + R'X \rightarrow ROR' + X^- RO−+R′X→ROR′+X−

例如：

(CH3)2CHONa+PhCH2Cl→PhCH2OCH(CH3)2(CH_3)_2CHONa + PhCH_2Cl \rightarrow PhCH_2OCH(CH_3)_2 (CH3​)2​CHONa+PhCH2​Cl→PhCH2​OCH(CH3​)2​

特点：

  • 本质为 SN2S_N2SN​2 反应
  • 适用于一级卤代烃较好
  • 位阻较大时反应效果变差

5.4 成酯反应

1. 无机含氧酸酯的生成

醇能与无机酸或其衍生物作用生成无机酸酯。

例如，丙三醇与硝酸反应可生成甘油三硝酸酯：

CH2OH−CHOH−CH2OH+3HNO3→ 10∘C H2SO4CH2ONO2−CHONO2−CH2ONO2+3H2O\mathrm{CH_2OH-CHOH-CH_2OH} + 3\mathrm{HNO_3} \xrightarrow[\ 10^\circ C\ ]{\mathrm{H_2SO_4}} \mathrm{CH_2ONO_2-CHONO_2-CH_2ONO_2} + 3\mathrm{H_2O} CH2​OH−CHOH−CH2​OH+3HNO3​H2​SO4​ 10∘C ​CH2​ONO2​−CHONO2​−CH2​ONO2​+3H2​O

又如，甲醇与硫酸反应，先生成硫酸氢甲酯，继续反应可生成硫酸二甲酯：

CH3OH+H2SO4⇌CH3OSO3H+H2O\mathrm{CH_3OH} + \mathrm{H_2SO_4} \rightleftharpoons \mathrm{CH_3OSO_3H} + \mathrm{H_2O} CH3​OH+H2​SO4​⇌CH3​OSO3​H+H2​O

2CH3OSO3H→减压蒸馏CH3OSO2OCH3+H2SO42\mathrm{CH_3OSO_3H} \xrightarrow{\mathrm{减压蒸馏}} \mathrm{CH_3OSO_2OCH_3} + \mathrm{H_2SO_4} 2CH3​OSO3​H减压蒸馏​CH3​OSO2​OCH3​+H2​SO4​

说明：

  • 硫酸二甲酯、硫酸二乙酯在工业上是重要的烷基化试剂
  • 其中硫酸二甲酯毒性较强

2. 有机羧酸酯的生成

醇与有机酸及其衍生物也能生成酯。

（1）与羧酸酯化

CH3OH+CH3COOH⇌CH3COOCH3+H2O\mathrm{CH_3OH} + \mathrm{CH_3COOH} \rightleftharpoons \mathrm{CH_3COOCH_3} + \mathrm{H_2O} CH3​OH+CH3​COOH⇌CH3​COOCH3​+H2​O

（2）与酰氯反应

CH3CH2OH+p-CH3C6H4SO2Cl→CH3CH2O ⁣− ⁣SO2C6H4CH3+HCl\mathrm{CH_3CH_2OH} + p\text{-}\mathrm{CH_3C_6H_4SO_2Cl} \rightarrow \mathrm{CH_3CH_2O\!-\!SO_2C_6H_4CH_3} + \mathrm{HCl} CH3​CH2​OH+p-CH3​C6​H4​SO2​Cl→CH3​CH2​O−SO2​C6​H4​CH3​+HCl

TsCl=p-CH3C6H4SO2ClTsCl = p\text{-}\mathrm{CH_3C_6H_4SO_2Cl} TsCl=p-CH3​C6​H4​SO2​Cl

生成的对甲苯磺酸酯（ROTsROTsROTs）很重要，因为它把原来难离去的 −OH-OH−OH 转化成了易离去基团 −OTs-OTs−OTs。

例如在醇的构型转化中，可先发生：

R−OH→TsCl/吡啶R−OTsR{-}OH \xrightarrow{TsCl/\text{吡啶}} R{-}OTs R−OHTsCl/吡啶​R−OTs

ROH+R′COOH⇌R′COOR+H2OROH + R'COOH \rightleftharpoons R'COOR + H_2O ROH+R′COOH⇌R′COOR+H2​O

5.5 卤代反应

醇中的羟基是强碱性基团、弱离去基团，因此通常要先活化，才能较容易转化为卤代烃。

1. 与氢卤酸反应

RCH2OH+HBr→RCH2Br+H2ORCH_2OH + HBr \rightarrow RCH_2Br + H_2O RCH2​OH+HBr→RCH2​Br+H2​O

例如苄醇与氢溴酸反应：

C6H5CH2OH+HBr→110∘CC6H5CH2Br+H2O\mathrm{C_6H_5CH_2OH} + \mathrm{HBr} \xrightarrow{110^\circ C} \mathrm{C_6H_5CH_2Br} + \mathrm{H_2O} C6​H5​CH2​OH+HBr110∘C​C6​H5​CH2​Br+H2​O

特点：

  • 先将 −OH-OH−OH 质子化，变成较好的离去基 H2OH_2OH2​O
  • 然后再发生取代反应

氢卤酸活性大小规律通常为：

HI>HBr>HClHI > HBr > HCl HI>HBr>HCl

2. 与卤化试剂反应

醇中的 −OH-OH−OH 是较差的离去基，通常需转化为更好的离去基后再发生取代反应。

（1）卤化磷

常用试剂：

  • PCl3PCl_3PCl3​
  • PCl5PCl_5PCl5​
  • PBr3PBr_3PBr3​
  • PI3PI_3PI3​（由 P+I2P + I_2P+I2​ 原位生成）

ROH→PX3/PX5RXROH \xrightarrow{PX_3 / PX_5} RX ROHPX3​/PX5​​RX

  • 适用于一级、二级醇
  • 通常为 SN2S_N2SN​2 机理
  • 构型发生反转

例如：

CH3CH2CH2CH2OH→ 165∘C PBr3CH3CH2CH2CH2Br(90%∼93%)\mathrm{CH_3CH_2CH_2CH_2OH} \xrightarrow[\ 165^\circ C\ ]{\mathrm{PBr_3}} \mathrm{CH_3CH_2CH_2CH_2Br} \quad (90\%\sim93\%) CH3​CH2​CH2​CH2​OHPBr3​ 165∘C ​CH3​CH2​CH2​CH2​Br(90%∼93%)

  • 生成气体 SO2SO_2SO2​、HClHClHCl，反应易于推动（不可逆）
  • 常用于制备氯代烃

  • 在吡啶存在下，多为 SN2S_N2SN​2
  • 无碱时可能经历离子机理
（3）Lucas 试剂

Lucas 试剂（ZnCl₂ + 浓HCl）的作用是：

1. 活化羟基

2. 提供卤素离子

5.6 脱水反应

醇在酸催化下加热可发生脱水反应，生成烯烃或醚。

（1）分子内脱水（生成烯烃）

  • 生成烯烃
  • 高温条件

六、酚的化学性质

6.1 酸碱性

（1）酚的酸性

酚表现为弱酸：

PhOH⇌PhO−+H+PhOH \rightleftharpoons PhO^- + H^+ PhOH⇌PhO−+H+

能与碱反应：

PhOH+NaOH→PhONa+H2OPhOH + NaOH \rightarrow PhONa + H_2O PhOH+NaOH→PhONa+H2​O

（2）酸性原因

酚失去质子后形成的阴离子中，负电荷可以在氧原子与芳环之间分散，使体系稳定性提高，因此更容易失去质子。

→ 酸性：酚 > 醇

（3）取代基影响

  • 吸电子基（−NO2-NO_2−NO2​、−CN-CN−CN）：增强酸性
  • 推电子基（−CH3-CH_3−CH3​、−OCH3-OCH_3−OCH3​）：减弱酸性

说明：

  • 邻、对位取代基可以通过共轭影响电子分布，作用更明显
  • 间位主要是诱导效应

6.2 酚羟基反应

（1）成盐反应

PhOH+NaOH→PhONa+H2OPhOH + NaOH \rightarrow PhONa + H_2O PhOH+NaOH→PhONa+H2​O

（2）醚化反应（重点）

酚不能通过分子间脱水生成醚

采用 Williamson 合成：

PhONa+R−X→PhOR+NaXPhONa + R-X \rightarrow PhOR + NaX PhONa+R−X→PhOR+NaX

条件与本质：

  • 先将酚转化为酚盐（提高亲核性）
  • 反应属于 SN2 机理
  • 一级卤代烃效果最好

原因：

  • 芳环上的碳为 sp2sp^2sp2 杂化，C–O 键较稳定
  • 难以形成类似醇脱水时的碳正离子中间体
  • 因此不能发生：

2ROH→H+ROR2ROH \xrightarrow{H^+} ROR 2ROHH+​ROR

（3）酯化反应

酚一般不与羧酸直接酯化，而采用：

PhOH+RCOCl→PhOCOR+HClPhOH + RCOCl \rightarrow PhOCO R + HCl PhOH+RCOCl→PhOCOR+HCl

或：

PhOH+(RCO)2O→PhOCOR+RCOOHPhOH + (RCO)_2O \rightarrow PhOCO R + RCOOH PhOH+(RCO)2​O→PhOCOR+RCOOH

原因：

  • 酚羟基亲核性较弱
  • 常规 Fischer 酯化难以进行

6.3 芳环上的取代反应

酚中的 −OH-OH−OH 使苯环电子云密度升高，从而：

  • 反应活性增强
  • 定位于邻位、对位

（1）卤代反应

PhOH+3Br2→2,4,6-三溴苯酚+3HBrPhOH + 3Br_2 \rightarrow 2,4,6\text{-三溴苯酚} + 3HBr PhOH+3Br2​→2,4,6-三溴苯酚+3HBr

特点：

  • 不需要催化剂
  • 反应速度快
  • 生成白色沉淀

（2）硝化反应

稀硝酸：

PhOH+HNO3→o-硝基酚+p-硝基酚PhOH + HNO_3 \rightarrow o\text{-硝基酚} + p\text{-硝基酚} PhOH+HNO3​→o-硝基酚+p-硝基酚

浓硝酸：

PhOH+3HNO3→2,4,6-三硝基苯酚+3H2OPhOH + 3HNO_3 \rightarrow 2,4,6\text{-三硝基苯酚} + 3H_2O PhOH+3HNO3​→2,4,6-三硝基苯酚+3H2​O

（3）磺化反应

PhOH+H2SO4→o/p-羟基苯磺酸PhOH + H_2SO_4 \rightarrow o/p\text{-羟基苯磺酸} PhOH+H2​SO4​→o/p-羟基苯磺酸

6.4 特征反应

（1）FeCl₃ 显色反应

PhOH+Fe3+→有色络合物（紫色）PhOH + Fe^{3+} \rightarrow \text{有色络合物（紫色）} PhOH+Fe3+→有色络合物（紫色）

用途：

  • 检验酚羟基

（2）Kolbe–Schmitt 反应

PhONa+CO2→加热,加压o-羟基苯甲酸PhONa + CO_2 \xrightarrow{加热,加压} o\text{-羟基苯甲酸} PhONa+CO2​加热,加压​o-羟基苯甲酸

说明：

  • 邻位产物为主
  • 工业制水杨酸

（3）Reimer–Tiemann 反应

PhOH+CHCl3+NaOH→o-羟基苯甲醛PhOH + CHCl_3 + NaOH \rightarrow o\text{-羟基苯甲醛} PhOH+CHCl3​+NaOH→o-羟基苯甲醛

条件：

  • 强碱环境
  • 生成二氯卡宾中间体
